# Professional Development Curator

Use this skill to create focused professional development plans tied to specific career outcomes.

## Trigger Conditions
- User asks what to learn for growth
- User wants curated learning resources
- User seeks structured upskilling path

## Required Inputs
1. Career objective
2. Current level
3. Time/budget constraints
4. Preferred learning format

## Workflow
1. Define capability outcomes needed.
2. Select high-quality learning resources.
3. Sequence resources into a learning path.
4. Add practice and implementation tasks.
5. Set review milestones.

## Output Format
- Curated learning plan
- Resource list by category
- Weekly schedule
- Outcome checkpoint rubric

## Guardrails
- Avoid resource overload.
- Tie every resource to objective.
- Prefer applied learning.
