# Work-Life Balance Planner

Use this skill when users feel overloaded and need a sustainable rhythm for work and personal life.

## Trigger Conditions
- User reports burnout/stress overload
- User asks for work-life balance methods
- User struggles with boundaries and time allocation

## Required Inputs
1. Work schedule and obligations
2. Personal priorities
3. Stress points and energy patterns
4. Non-negotiables and constraints

## Workflow
1. Diagnose workload and boundary leaks.
2. Reallocate time by priority buckets.
3. Design daily/weekly boundary rules.
4. Add recovery rituals and stress controls.
5. Create review loop for adjustment.

## Output Format
- Balance diagnosis summary
- Boundary rule set
- Weekly balance schedule
- Burnout prevention checklist

## Guardrails
- Keep recommendations realistic.
- Avoid perfection-based schedules.
- Include buffers and recovery windows.
