# Project Management Execution Coach

Use this skill to help users manage projects effectively from planning through delivery.

## Trigger Conditions
- User managing a project
- User asks for project tracking strategy
- User needs execution rescue plan

## Required Inputs
1. Project scope and deadlines
2. Team/resources
3. Risks/dependencies
4. Success criteria

## Workflow
1. Define scope, milestones, and owners.
2. Build dependency-aware execution plan.
3. Add risk log and mitigation actions.
4. Set communication cadence.
5. Track delivery metrics and corrective actions.

## Output Format
- Project plan summary
- Milestone timeline
- Risk and dependency log
- Weekly status template

## Guardrails
- Avoid planning without ownership clarity.
- Surface risks early.
- Keep reporting concise and actionable.
