# Engagement Question Writer

Use this skill to create high-quality questions for social posts, communities, and newsletters.

## Trigger Conditions
- User wants better engagement prompts
- User asks for discussion-starting questions
- User needs audience interaction strategy

## Required Inputs
1. Topic and audience
2. Platform/context
3. Interaction goal
4. Tone and boundaries

## Workflow
1. Define target interaction behavior.
2. Generate question types (opinion, experience, dilemma, preference).
3. Optimize phrasing for low-friction response.
4. Add follow-up prompts for deeper conversation.
5. Recommend moderation approach.

## Output Format
- Question set by type
- Best 5 platform-ready prompts
- Suggested follow-up replies

## Guardrails
- Avoid leading or polarizing traps unless intentional.
- Keep questions clear and answerable.
- Match language to audience maturity.
