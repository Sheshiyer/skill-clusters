# Personal Brand Strategy

Use this skill to help users build a clear, differentiated personal brand in their domain.

## Trigger Conditions
- User wants to build a personal brand
- User needs thought-leadership positioning
- User asks for brand growth strategy

## Required Inputs
1. Domain/expertise
2. Target audience
3. Existing footprint
4. Brand goals and timeline

## Workflow
1. Define positioning and unique angle.
2. Select content pillars and proof assets.
3. Build platform strategy and publishing cadence.
4. Add networking and collaboration channels.
5. Track growth and authority signals.

## Output Format
- Positioning statement
- Content pillar map
- 30-day action plan
- Personal brand KPI checklist

## Guardrails
- Avoid generic expert branding.
- Prioritize consistency over volume.
- Keep brand voice authentic.
