# Passive Income Planner

Use this skill to evaluate and plan passive-income opportunities with practical constraints.

## Trigger Conditions
- User asks about passive income options
- User wants secondary revenue streams
- User needs prioritization among options

## Required Inputs
1. Capital/time available
2. Skills and assets
3. Risk tolerance
4. Income goals and timeline

## Workflow
1. Identify suitable passive-income models.
2. Evaluate setup complexity and risk.
3. Prioritize options by fit.
4. Build implementation roadmap.
5. Track performance and reinvestment plan.

## Output Format
- Option comparison table
- Recommended path(s)
- 90-day implementation plan
- Risk and monitoring checklist

## Guardrails
- Avoid “easy money” framing.
- Clarify active effort required upfront.
- State assumptions and uncertainties.
