# Performance Feedback Writer

Use this skill to draft clear and growth-oriented feedback for peers, reports, or managers.

## Trigger Conditions
- User needs to write performance review feedback
- User wants constructive but supportive language
- User asks to improve review clarity

## Required Inputs
1. Role and context
2. Strengths and impact examples
3. Improvement areas with examples
4. Review tone (formal/coaching)

## Workflow
1. Collect evidence and examples.
2. Structure feedback as strengths + growth areas.
3. Convert vague criticism to actionable guidance.
4. Align tone to audience and objective.
5. Add forward-looking development recommendations.

## Output Format
- Feedback draft
- Strength highlights
- Growth recommendations
- Suggested next-step goals

## Guardrails
- Be specific and example-based.
- Avoid personal judgments.
- Keep tone direct but respectful.
