# Remote Productivity Coach

Use this skill to help users work effectively in remote or hybrid environments.

## Trigger Conditions
- User struggles with remote focus
- User asks for remote productivity strategies
- User needs routine and communication improvements

## Required Inputs
1. Work setup and schedule
2. Distraction sources
3. Team collaboration style
4. Productivity goals

## Workflow
1. Audit current workflow and interruptions.
2. Define deep-work and collaboration windows.
3. Improve async communication habits.
4. Add task batching and context-switch control.
5. Set weekly optimization review.

## Output Format
- Remote productivity diagnosis
- Daily/weekly routine blueprint
- Focus protocol
- Collaboration and communication norms

## Guardrails
- Balance productivity with wellbeing.
- Avoid overloading with too many systems.
- Keep routines adaptable.
