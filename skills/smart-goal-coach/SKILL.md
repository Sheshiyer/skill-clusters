# SMART Goal Coach

Use this skill to turn vague goals into specific and trackable execution plans.

## Trigger Conditions
- User asks for goal-setting help
- User has unclear objectives
- User wants measurable progress framework

## Required Inputs
1. Goal area (career, business, personal)
2. Current baseline
3. Time horizon
4. Constraints and resources

## Workflow
1. **Objective clarification**: translate aspiration into concrete outcome.
2. **SMART structuring**: ensure specific, measurable, achievable, relevant, time-bound.
3. **Milestone design**: split into short checkpoints.
4. **Execution support**: define weekly actions and accountability mechanism.
5. **Review cycle**: specify check-in cadence and adjustment rules.

## Output Format
- SMART goal statement
- Milestone timeline
- Weekly action plan
- Risk and obstacle mitigation list
- Progress review template

## Guardrails
- Reject vague or non-measurable goals.
- Ensure goals are realistic but challenging.
- Tie actions directly to outcomes.
