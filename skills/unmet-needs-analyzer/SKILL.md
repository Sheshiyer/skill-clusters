# Unmet Needs Analyzer

Use this skill to uncover category-level unmet needs and convert insights into strategic opportunities.

## Trigger Conditions
- User asks for market gaps or unmet needs
- User researching new product opportunities
- User needs demand-side insight generation

## Required Inputs
1. Category/market definition
2. Target audience context
3. Known offerings/competitors (optional)
4. Geography or segment constraints (optional)

## Workflow
1. **Need framing**: define the problem space and boundaries.
2. **Friction mapping**: identify where existing solutions fail users.
3. **Need clustering**: group unmet needs by severity and frequency.
4. **Opportunity translation**: map needs to potential offers/initiatives.
5. **Prioritization**: rank opportunities by impact and feasibility.

## Output Format
- Top unmet needs list with titles
- Why each need remains unsolved
- Opportunity concepts tied to each need
- Prioritized opportunity shortlist

## Guardrails
- Distinguish true unmet needs from preferences.
- Avoid generic claims without reasoning.
- State assumptions and confidence level.
