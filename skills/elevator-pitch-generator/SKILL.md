# Elevator Pitch Generator

Use this skill to produce short persuasive pitches for products, startups, or personal positioning.

## Trigger Conditions
- User asks for an elevator pitch
- User needs concise verbal positioning
- User wants variants for investor/customer/networking contexts

## Required Inputs
1. Offering/person summary
2. Audience type
3. Problem solved
4. Differentiator and proof

## Workflow
1. Extract core value in one sentence.
2. Add audience-relevant problem framing.
3. Include differentiation and proof.
4. Generate 15s/30s/60s variants.
5. Polish for spoken delivery.

## Output Format
- 15-second pitch
- 30-second pitch
- 60-second pitch
- Delivery tips

## Guardrails
- Avoid jargon-heavy language.
- Lead with value, not feature list.
- Keep claims credible and specific.
