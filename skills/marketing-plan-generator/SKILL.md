# Marketing Plan Generator

Use this skill when the user needs a practical marketing plan for a product, launch, or growth objective.

## Trigger Conditions
- User asks for a marketing plan
- User wants channel selection and campaign structure
- User needs tactical go-to-market steps

## Required Inputs
1. Product/service and value proposition
2. Target audience
3. Budget/time constraints
4. Goal (awareness, leads, sales, retention)

## Workflow
1. **Goal framing**: translate business goal into measurable marketing objective.
2. **Audience strategy**: define segments and core messages.
3. **Channel plan**: prioritize channels by fit, speed, cost, measurability.
4. **Execution roadmap**: weekly action plan with owners and deliverables.
5. **Measurement**: KPI tree and review cadence.

## Output Format
- 1-page strategy summary
- Channel priority matrix
- 4-12 week action plan
- KPI dashboard definition
- Iteration recommendations

## Guardrails
- Avoid generic channel recommendations without audience rationale.
- Prefer specific actions and timelines.
- Tie every tactic to an objective.
