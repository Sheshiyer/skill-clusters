# Career Advancement Planner

Use this skill when the user wants to grow within their current profession, move toward a promotion, or increase strategic influence.

## Trigger Conditions
- User asks about growth in current role
- User wants promotion readiness planning
- User asks how to become more valuable at work

## Required Inputs
Collect these before planning:
1. Current role and level
2. Target role/level and timeline
3. Current strengths and weaknesses
4. Org context (team size, manager expectations, promotion criteria if known)

## Workflow
1. **Clarify destination**: define target scope, title, and expected outcomes.
2. **Gap analysis**: compare current capability vs target capability in skills, influence, outcomes, and leadership.
3. **90-day plan**: create actions grouped as skill-building, visibility, and impact projects.
4. **Risk handling**: identify blockers and mitigation actions.
5. **Progress loop**: set weekly and monthly review checkpoints.

## Output Format
Return:
- Career objective statement
- Capability gap table (current vs target)
- 30/60/90-day plan with measurable milestones
- Stakeholder communication plan
- Tracking checklist

## Guardrails
- Prefer practical actions over generic motivation.
- Keep recommendations realistic for available time and context.
- Avoid one-size-fits-all promotion advice.
