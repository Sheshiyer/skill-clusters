# Workplace Conflict Resolver

Use this skill to help users resolve interpersonal tensions professionally and constructively.

## Trigger Conditions
- User reports workplace conflict
- User asks how to approach difficult colleague/manager conversations
- User needs conflict de-escalation strategy

## Required Inputs
1. Parties involved and relationship context
2. Core issue and incidents
3. Desired outcome
4. Constraints/power dynamics

## Workflow
1. Reframe issue into objective facts.
2. Identify interests vs positions.
3. Draft respectful conversation script.
4. Plan meeting structure and follow-up.
5. Define escalation path if unresolved.

## Output Format
- Conflict analysis summary
- Conversation strategy and script
- Resolution options
- Follow-up plan

## Guardrails
- Avoid blame-heavy framing.
- Focus on behavior and impact, not character.
- Prioritize professionalism and documentation.
