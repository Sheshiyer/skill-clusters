# Mission Vision Refiner

Use this skill to sharpen mission and vision so they guide strategy and inspire stakeholders.

## Trigger Conditions
- User wants to refine mission/vision
- User has unclear strategic narrative
- User needs concise company direction statements

## Required Inputs
1. Current mission/vision (if any)
2. Business model and audience
3. Long-term aspiration
4. Current strategic focus

## Workflow
1. Diagnose clarity and alignment gaps.
2. Redraft mission (present purpose).
3. Redraft vision (future aspiration).
4. Validate against strategy and audience.
5. Produce variants for internal/external use.

## Output Format
- Refined mission statement
- Refined vision statement
- Rationale for wording
- Usage guidance

## Guardrails
- Avoid buzzword-heavy statements.
- Keep mission actionable and vision aspirational.
- Ensure consistency with actual direction.
