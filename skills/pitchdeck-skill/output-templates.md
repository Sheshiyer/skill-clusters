# Output Templates

Templates for all files generated by the skill.

## Overview.md Template

```markdown
# {Company Name} — Investor Experience

> {One-line pitch: the "bar napkin" version}

## Executive Summary

{2-3 paragraph summary covering: what it is, who it's for, why now, key traction, the ask}

## The Narrative Arc

The pitch flows as a 10-section emotional journey:

| # | Section | Duration | Emotional Beat | Key Message |
|---|---------|----------|----------------|-------------|
| 1 | Hero | 0-5s | Intrigue | {Hook statement} |
| 2 | Problem | 5-20s | Pain | {Problem articulation} |
| 3 | Solution | 20-40s | Relief | {How we solve it} |
| 4 | Traction | 40-60s | Trust | {Proof points} |
| 5 | Market | 60-80s | Excitement | {Opportunity size} |
| 6 | Model | 80-100s | Confidence | {Revenue mechanics} |
| 7 | Competition | 100-115s | Clarity | {Our advantage} |
| 8 | Team | 115-130s | Credibility | {Why we win} |
| 9 | Roadmap | 130-145s | Belief | {What's next} |
| 10 | Ask | 145-160s | Action | {Investment terms} |

## Key Messages

### Primary Value Proposition
{One sentence: what + for whom + unique benefit}

### Supporting Messages
1. {Differentiator 1}
2. {Differentiator 2}
3. {Differentiator 3}

### Proof Points
- {Traction metric 1}
- {Traction metric 2}
- {Traction metric 3}

## Target Investor Profile

- **Stage:** {Seed / Series A / B}
- **Check Size:** {$X - $Y}
- **Sectors:** {Relevant focus areas}
- **Geography:** {Target markets}

## Emotional Journey Map

| Section | Investor Emotion | Design Intent | Visual Elements |
|---------|------------------|---------------|-----------------|
| Hero | Curiosity, intrigue | Bold first impression | TextType animation, gradient mesh |
| Problem | Recognition, frustration | Build tension | Dark tones, pain statistics |
| Solution | Relief, hope | Release tension | Product reveal, bright accent |
| Traction | Trust, validation | Build credibility | Animated counters, logos |
| Market | Excitement, greed | Show opportunity | TAM visualization |
| Model | Confidence | Demonstrate understanding | Clean data viz |
| Competition | Clarity | Show strategic thinking | Position matrix |
| Team | Trust, credibility | Human connection | Team cards |
| Roadmap | Belief, anticipation | Show execution ability | Timeline stepper |
| Ask | Confidence to act | Clear call to action | Prominent CTA |

## Potential Objections & Rebuttals

| Objection | Rebuttal |
|-----------|----------|
| {Common concern 1} | {Response} |
| {Common concern 2} | {Response} |
| {Common concern 3} | {Response} |

## Research Sources

- {Source 1: Market data}
- {Source 2: Industry report}
- {Source 3: Competitor info}
```

## DesignSpec.md Template

```markdown
# Design Specification

## Design Direction

**Aesthetic:** {e.g., "Void-to-light transformation journey"}
**Industry:** {e.g., "B2B SaaS"}
**Mood:** {e.g., "Professional trust with innovative edge"}

## Color System

### Primary Palette

| Token | Hex | RGB | Usage |
|-------|-----|-----|-------|
| void | #060010 | 6, 0, 16 | Primary background |
| surface | #222222 | 34, 34, 34 | Card backgrounds |
| glass | rgba(255,255,255,0.05) | - | Glass elements |
| accent-primary | {#5227FF} | - | CTAs, highlights |
| accent-secondary | {#22D3EE} | - | Secondary accents |
| text-primary | #FFFFFF | 255, 255, 255 | Headlines |
| text-secondary | #A3A3A3 | 163, 163, 163 | Body copy |

### Gradients

```css
--gradient-hero: radial-gradient(ellipse at 20% 80%, rgba({r},{g},{b},0.15) 0%, transparent 50%), radial-gradient(ellipse at 80% 20%, rgba({r},{g},{b},0.1) 0%, transparent 50%), #060010;

--gradient-accent: linear-gradient(135deg, {accent-primary}, {accent-secondary});
```

## Typography

| Element | Font | Weight | Size | Line Height |
|---------|------|--------|------|-------------|
| Hero headline | Space Grotesk | 700 | clamp(3rem, 8vw, 5rem) | 1.0 |
| Section title | Space Grotesk | 700 | clamp(2rem, 5vw, 3rem) | 1.1 |
| Card heading | Space Grotesk | 600 | 1.5rem | 1.2 |
| Body | Inter | 400 | 1.125rem | 1.6 |
| Metrics | JetBrains Mono | 700 | clamp(3rem, 6vw, 4rem) | 1.0 |
| Caption | Inter | 500 | 0.875rem | 1.4 |

## Component Specifications

### GlassCard
```css
background: rgba(255, 255, 255, 0.05);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.1);
border-radius: 24px;
box-shadow: 0 8px 32px rgba(0,0,0,0.12), inset 0 0 0 1px rgba(255,255,255,0.05);
```

### Hover States
```css
transform: translateY(-4px);
background: rgba(255, 255, 255, 0.08);
box-shadow: 0 16px 48px rgba(0,0,0,0.2);
```

## Animation Choreography

### Easing
```css
--ease-smooth: cubic-bezier(0.25, 0.1, 0.25, 1);
--ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
```

### Timing
| Animation | Duration | Stagger |
|-----------|----------|---------|
| Section reveal | 600ms | - |
| Card fade in | 500ms | 100ms |
| Counter | 2000ms | - |
| Hover | 300ms | - |

### Scroll Triggers
| Section | Trigger | Animation |
|---------|---------|-----------|
| Hero | Immediate | TextType + CircularText |
| Problem | 20% viewport | Stagger cards up |
| Traction | 30% viewport | Counter animation |
| Market | 40% viewport | Circle scale |

## Responsive Breakpoints

| Breakpoint | Width | Grid | Font Scale |
|------------|-------|------|------------|
| Mobile | <640px | 1 col | 0.85x |
| Tablet | 640-1024px | 2 col | 0.95x |
| Desktop | >1024px | 3 col | 1x |

## Accessibility

- Reduced motion: respect `prefers-reduced-motion`
- Contrast: minimum 4.5:1 for body text
- Focus: visible ring on interactive elements
```

## Tasks.md Template

```markdown
# Build Tasks

## Project: {Company Name} Pitch Experience

### Phase 0: Setup

```bash
# Initialize project
bun create vite {company-slug}-pitch --template react
cd {company-slug}-pitch

# Install dependencies
bun add framer-motion gsap lucide-react recharts
bun add -d tailwindcss postcss autoprefixer
bunx tailwindcss init -p
```

- [ ] Configure Tailwind with design tokens
- [ ] Set up font imports
- [ ] Create folder structure:
  ```
  src/
  ├── components/
  │   ├── ui/           # GlassCard, GradientText, etc.
  │   └── sections/     # Hero, Problem, Solution, etc.
  ├── styles/
  │   └── tokens.css
  └── data/
      └── content.js
  ```

### Phase 1: Design System

- [ ] Create `tokens.css` with all CSS variables
- [ ] Build base components:
  - [ ] GlassCard
  - [ ] GradientText
  - [ ] FadeInView
  - [ ] StaggerContainer
- [ ] Test in isolation

### Phase 2: Animation Components

- [ ] TextType (typewriter)
- [ ] CircularText (rotating tagline)
- [ ] AnimatedCounter
- [ ] ScrollVelocity (marquee)
- [ ] ProgressStepper

### Phase 3: Section Components

| Section | Priority | Components Used |
|---------|----------|-----------------|
| Hero | P0 | TextType, CircularText |
| Problem | P0 | GlassCard, StaggerContainer |
| Solution | P0 | FadeInView, GlassCard |
| Traction | P1 | AnimatedCounter, ScrollVelocity |
| Market | P1 | Recharts, FadeInView |
| Model | P1 | GlassCard grid |
| Competition | P2 | Position matrix |
| Team | P2 | Team cards |
| Roadmap | P2 | ProgressStepper |
| Ask | P0 | CTA button |

### Phase 4: Content Integration

- [ ] Import copy from `content/copy-blocks.md`
- [ ] Import data from `research/`
- [ ] Verify all sections populated
- [ ] Review narrative flow

### Phase 5: Polish

- [ ] Add reduced motion support
- [ ] Mobile responsive pass
- [ ] Performance audit
- [ ] Cross-browser test

### Phase 6: Deploy

- [ ] Build: `bun run build`
- [ ] Deploy to Vercel/Netlify
- [ ] Generate shareable link

## QA Checklist

- [ ] All animations smooth (60fps)
- [ ] Text readable at all sizes
- [ ] Interactive elements have hover states
- [ ] Metrics animate on scroll
- [ ] Mobile layout correct
- [ ] Lighthouse score >90
```
