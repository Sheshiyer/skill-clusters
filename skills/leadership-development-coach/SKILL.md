# Leadership Development Coach

Use this skill to help users strengthen leadership capability across people, process, and strategic execution.

## Trigger Conditions
- User stepping into leadership role
- User wants to improve team leadership
- User asks for leadership growth roadmap

## Required Inputs
1. Current role and scope of leadership
2. Team context and key challenges
3. Leadership goals and timeline
4. Feedback currently received (if available)

## Workflow
1. **Leadership baseline**: assess strengths and recurring gaps.
2. **Capability focus**: prioritize communication, delegation, coaching, decision-making.
3. **Behavioral plan**: define concrete weekly leadership behaviors.
4. **Feedback loop**: establish signal collection and reflection routine.
5. **Scaling impact**: create systems for team autonomy and accountability.

## Output Format
- Leadership maturity snapshot
- 30/60/90-day development plan
- Weekly behavior commitments
- Team communication framework
- Progress indicators

## Guardrails
- Prefer observable behaviors over vague traits.
- Keep plans context-specific.
- Emphasize consistency over intensity.
