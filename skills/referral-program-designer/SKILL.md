# Referral Program Designer

Use this skill to create referral systems that encourage advocacy and measurable growth.

## Trigger Conditions
- User wants a referral program
- User asks how to increase word-of-mouth
- User needs incentive design help

## Required Inputs
1. Product and customer profile
2. Current retention/satisfaction signals
3. Incentive budget
4. Referral channel options

## Workflow
1. Define referral objective and audience.
2. Select reward model (single/double-sided).
3. Design referral journey and UX touchpoints.
4. Add anti-abuse and quality checks.
5. Set launch and optimization plan.

## Output Format
- Referral model recommendation
- Incentive structure
- User flow outline
- Success metrics dashboard

## Guardrails
- Ensure incentive aligns with LTV economics.
- Keep mechanics simple for users.
- Include abuse prevention from day one.
