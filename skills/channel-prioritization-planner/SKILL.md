# Channel Prioritization Planner

Use this skill when the user needs to decide which marketing channels to invest in first.

## Trigger Conditions
- User asks which channels matter most
- User has limited budget/time for marketing
- User wants ranked channel strategy

## Required Inputs
1. Product/category and target audience
2. Budget and timeline constraints
3. Existing channel performance (if any)
4. Primary objective (awareness, leads, revenue)

## Workflow
1. **Criteria definition**: audience fit, reach, speed, cost, control, measurability.
2. **Channel scoring**: evaluate candidate channels against criteria.
3. **Prioritization**: rank channels and define role for top tiers.
4. **Activation plan**: define first tests and expected outcomes.
5. **Optimization loop**: set review cadence and pivot criteria.

## Output Format
- Ranked channel table with rationale
- Top 3 channel activation playbook
- Budget allocation suggestion
- KPI and review cadence

## Guardrails
- Don’t recommend channels without audience logic.
- Avoid spreading thin across too many channels.
- Tie spend to learning milestones.
