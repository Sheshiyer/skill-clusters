# Newsletter PAS Writer

Use this skill to write newsletter sections with Problem-Agitate-Solve clarity and conversion intent.

## Trigger Conditions
- User wants newsletter intro/copy
- User asks to apply PAS framework
- User needs engagement-oriented email writing

## Required Inputs
1. Topic and audience pain
2. Newsletter goal
3. Brand tone
4. CTA target

## Workflow
1. Define core problem in audience language.
2. Agitate consequences with empathy.
3. Present practical solution pathway.
4. Align closing CTA.
5. Refine for email readability.

## Output Format
- PAS intro draft
- Alternative tone variants
- CTA options

## Guardrails
- Avoid manipulative fear tactics.
- Keep solution concrete and believable.
- Maintain brand-consistent voice.
