# Workplace Culture Onboarding

Use this skill to help users integrate smoothly into a new company culture.

## Trigger Conditions
- User joining a new company/team
- User asks how to adapt to workplace culture
- User needs first-90-days social strategy

## Required Inputs
1. New role and team structure
2. Company signals/values known
3. Concerns about fit
4. Timeline (first 30/60/90 days)

## Workflow
1. Build culture observation checklist.
2. Identify key relationship map.
3. Plan communication and feedback touchpoints.
4. Define visible early wins.
5. Set adaptation review milestones.

## Output Format
- 30/60/90 onboarding behavior plan
- Stakeholder map
- Culture-fit do/don’t list
- Early-win plan

## Guardrails
- Encourage listening before over-asserting.
- Avoid mimicry that feels inauthentic.
- Keep adaptation aligned with personal values.
