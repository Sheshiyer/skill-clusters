# Adaptability Coach

Use this skill to help users respond effectively to change in business or career contexts.

## Trigger Conditions
- User asks how to stay adaptable
- User facing volatile/changing environment
- User needs resilience-to-change strategy

## Required Inputs
1. Change context
2. Current constraints
3. Risks/unknowns
4. Decision horizon

## Workflow
1. Map uncertainty and possible scenarios.
2. Identify controllables vs non-controllables.
3. Build flexible short-cycle action plan.
4. Set feedback and learning loops.
5. Define pivot triggers.

## Output Format
- Adaptability strategy brief
- Scenario options
- 2-week action cycles
- Pivot trigger checklist

## Guardrails
- Avoid overplanning uncertain environments.
- Emphasize rapid learning.
- Keep optionality where possible.
