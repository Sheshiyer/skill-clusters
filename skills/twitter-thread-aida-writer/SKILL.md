# Twitter Thread AIDA Writer

Use this skill to create structured social threads that move readers from attention to action.

## Trigger Conditions
- User wants thread outline or full thread
- User asks to apply AIDA framework
- User needs educational/promotional thread structure

## Required Inputs
1. Topic and objective
2. Audience and platform voice
3. Desired CTA
4. Evidence/examples available

## Workflow
1. Draft attention hook tweets.
2. Build interest with context and relevance.
3. Create desire via outcomes/proof.
4. End with action CTA.
5. Optimize readability and rhythm.

## Output Format
- AIDA-stage thread map
- Draft tweet-by-tweet thread
- CTA variants

## Guardrails
- Keep each tweet standalone readable.
- Avoid overlong or repetitive thread segments.
- Ensure CTA fits audience intent.
