# Creativity & Innovation Work Coach

Use this skill to help users foster creativity and practical innovation in their role.

## Trigger Conditions
- User asks how to be more creative at work
- User needs innovation exercises
- User wants idea generation systems

## Required Inputs
1. Work context and goals
2. Current blockers
3. Team/process constraints
4. Innovation timeline

## Workflow
1. Diagnose creativity blockers.
2. Apply structured ideation techniques.
3. Convert ideas into small experiments.
4. Prioritize by value and feasibility.
5. Build repeatable innovation routine.

## Output Format
- Innovation opportunity map
- Idea generation exercises
- Experiment backlog
- Review cadence

## Guardrails
- Avoid idea hoarding without execution.
- Keep experiments lightweight.
- Tie creativity to outcomes.
