# Listicle Idea Generator

Use this skill to ideate list-style content topics for social, newsletter, and blogs.

## Trigger Conditions
- User asks for listicle ideas
- User needs easy-to-consume content concepts
- User wants highly shareable formats

## Required Inputs
1. Topic domain
2. Target audience
3. Platform format
4. Objective (reach/leads/authority)

## Workflow
1. Identify high-interest subtopics.
2. Generate listicle angles by intent.
3. Score for novelty and practicality.
4. Expand best ideas into outlines.
5. Suggest publishing cadence.

## Output Format
- 10+ listicle concepts
- Top 3 detailed outlines
- Recommended publishing order

## Guardrails
- Avoid repetitive “generic tips” content.
- Prioritize actionable and specific angles.
- Tailor list lengths to platform norms.
