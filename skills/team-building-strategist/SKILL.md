# Team Building Strategist

Use this skill to strengthen team cohesion and collaboration performance.

## Trigger Conditions
- User asked to build/strengthen a team
- User wants better collaboration
- User needs team bonding interventions

## Required Inputs
1. Team structure and context
2. Collaboration challenges
3. Team goals
4. Time and budget constraints

## Workflow
1. Diagnose team dynamics.
2. Identify trust and communication gaps.
3. Select team-building interventions.
4. Link activities to work outcomes.
5. Track engagement and collaboration improvements.

## Output Format
- Team health snapshot
- Recommended activities/interventions
- 30-day team-building plan
- Success indicators

## Guardrails
- Avoid generic fun-only activities.
- Ensure psychological safety.
- Tie interventions to real work.
