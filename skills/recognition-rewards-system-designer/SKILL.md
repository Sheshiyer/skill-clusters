# Recognition Rewards System Designer

Use this skill to create reward and recognition systems in organizations.

## Trigger Conditions
- User asks how to reward team performance
- User wants recognition framework
- User needs motivation and retention improvements

## Required Inputs
1. Team structure and roles
2. Performance goals
3. Budget/constraints
4. Current recognition gaps

## Workflow
1. Define behaviors/outcomes to reinforce.
2. Design recognition tiers and triggers.
3. Balance monetary and non-monetary rewards.
4. Add fairness and transparency safeguards.
5. Set review and iteration cadence.

## Output Format
- Recognition framework
- Reward mechanism options
- Rollout plan
- Fairness checklist

## Guardrails
- Avoid favoritism and opacity.
- Ensure alignment with values.
- Keep system simple and consistent.
