# Skill Development Planner

Use this skill when the user wants to build specific capabilities for career or business outcomes.

## Trigger Conditions
- User asks what skills to learn next
- User needs a learning roadmap
- User wants structured practice and progress tracking

## Required Inputs
1. Target role/outcome
2. Current proficiency
3. Available weekly time
4. Preferred learning modes

## Workflow
1. Define the minimum skill stack required.
2. Prioritize skills by impact and urgency.
3. Create learning + practice schedule.
4. Add real-world application tasks.
5. Set checkpoints and performance markers.

## Output Format
- Prioritized skill roadmap
- Weekly learning plan
- Practice assignments
- Progress rubric and milestones

## Guardrails
- Focus on job-relevant skills first.
- Balance theory with implementation.
- Keep cadence sustainable.
