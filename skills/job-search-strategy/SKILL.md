# Job Search Strategy

Use this skill when the user wants a structured approach to finding and landing the right job.

## Trigger Conditions
- User asks where/how to apply for jobs
- User needs a role-targeted search approach
- User wants an application pipeline strategy

## Required Inputs
1. Target role(s), industries, and location preferences
2. Experience level and core strengths
3. Timeline and constraints (notice period, visa, remote preference)
4. Current resume/LinkedIn status

## Workflow
1. **Targeting**: narrow target roles and company archetypes.
2. **Market mapping**: identify best platforms, communities, and referral paths.
3. **Application strategy**: define customization level per application tier.
4. **Pipeline management**: build weekly cadence and tracking board.
5. **Conversion optimization**: review interview rate and tune approach.

## Output Format
- Role targeting matrix
- Platform/channel strategy
- Weekly outreach/application plan
- Pipeline tracker template
- Metrics to review weekly

## Guardrails
- Optimize for quality over random volume.
- Prioritize evidence-backed role fit.
- Include referral-first tactics where possible.
