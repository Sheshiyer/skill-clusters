# Audience Segmentation Generator

Use this skill to build actionable audience segments for marketing, product, and messaging decisions.

## Trigger Conditions
- User asks to define personas or segments
- User needs deeper customer understanding
- User asks for audience-based channel/messaging strategy

## Required Inputs
1. Category/product context
2. Existing customer clues (if any)
3. Goal of segmentation (acquisition, retention, product fit)

## Workflow
1. **Segmentation objective**: clarify what decisions segmentation should support.
2. **Need-based clustering**: segment by jobs, motivations, and barriers.
3. **Behavior lens**: add channel, content, and buying behavior traits.
4. **Prioritization**: rank segments by opportunity and fit.
5. **Activation**: map each segment to messaging and initiatives.

## Output Format
- 5-10 segment profiles
- Segment need statement + pain points
- Priority score/rationale
- Recommended positioning and channel tactics per segment

## Guardrails
- Avoid demographic-only segmentation.
- Keep segments mutually distinct and decision-useful.
- Be explicit where assumptions are used.
