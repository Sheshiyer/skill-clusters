# Portfolio Builder

Use this skill to help users build a compelling portfolio for job, freelance, or client opportunities.

## Trigger Conditions
- User asks what to include in portfolio
- User needs portfolio improvement
- User wants role-specific project presentation

## Required Inputs
1. Target role/audience
2. Existing projects/work samples
3. Proof points (metrics/testimonials)
4. Preferred format (web/PDF/Notion)

## Workflow
1. Define portfolio objective by audience.
2. Select strongest evidence-based projects.
3. Structure each case study (problem-action-result).
4. Improve narrative, visuals, and credibility signals.
5. Add CTA and maintenance plan.

## Output Format
- Portfolio structure
- Case study template
- Project selection shortlist
- Improvement checklist

## Guardrails
- Prioritize quality over number of projects.
- Use measurable outcomes whenever possible.
- Avoid clutter and vague claims.
