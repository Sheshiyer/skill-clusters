# Sales Funnel Designer

Use this skill to create or improve sales funnels that move prospects from discovery to conversion.

## Trigger Conditions
- User asks for funnel design
- User has traffic but low conversions
- User wants stage-wise sales process

## Required Inputs
1. Product/service and offer details
2. Target audience and buying behavior
3. Current funnel (if any)
4. Goal metric (leads, demos, purchases)

## Workflow
1. **Funnel mapping**: define stages and desired transitions.
2. **Stage messaging**: align message with buyer intent at each stage.
3. **Asset planning**: specify content/offers/CTAs per stage.
4. **Friction reduction**: identify leaks and optimize flow.
5. **Measurement plan**: choose stage conversion metrics.

## Output Format
- Funnel architecture diagram (textual)
- Stage-by-stage copy and CTA guidance
- Conversion optimization recommendations
- KPI and experiment plan

## Guardrails
- Avoid one-message-fits-all funnels.
- Keep each stage tied to one clear next action.
- Include nurture strategy for non-buyers.
