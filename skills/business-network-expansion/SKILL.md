# Business Network Expansion

Use this skill to grow a business owner’s network for opportunities, partnerships, and market access.

## Trigger Conditions
- User asks to expand business network
- User seeks partnerships/referrals
- User needs networking strategy for growth

## Required Inputs
1. Industry and growth goals
2. Current network map
3. Target connection types
4. Outreach channels

## Workflow
1. Define networking outcomes.
2. Build target relationship map.
3. Create outreach and follow-up sequences.
4. Add value-first engagement strategy.
5. Track conversations and outcomes.

## Output Format
- Network expansion plan
- Target contact categories
- Outreach templates
- Follow-up cadence

## Guardrails
- Prioritize trust over volume.
- Personalize outreach.
- Keep long-term reciprocity mindset.
