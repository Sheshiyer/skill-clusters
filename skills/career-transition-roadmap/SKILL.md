# Career Transition Roadmap

Use this skill to help users move into a new field with a structured and realistic transition plan.

## Trigger Conditions
- User wants to switch careers
- User asks how to transition without starting from zero
- User needs a roadmap for new role entry

## Required Inputs
1. Current role/background
2. Target role/industry
3. Timeline and constraints
4. Existing transferable strengths

## Workflow
1. Define transition target and success criteria.
2. Map transferable skills and credibility gaps.
3. Build 30/60/90-day transition plan.
4. Add portfolio/network/job-search components.
5. Set review checkpoints and pivot signals.

## Output Format
- Transition objective
- Transferable-skill map
- Gap-closure action plan
- 90-day roadmap
- Weekly progress checklist

## Guardrails
- Keep plan grounded in user constraints.
- Prefer demonstrable proof-building actions.
- Avoid generic “learn everything” advice.
