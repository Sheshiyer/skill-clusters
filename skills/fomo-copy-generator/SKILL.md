# FOMO Copy Generator

Use this skill to create urgency-driven copy that prompts timely action.

## Trigger Conditions
- User wants scarcity/urgency copy
- User promotes event, launch, or limited offer
- User asks for FOMO-based messaging

## Required Inputs
1. Offer/event details
2. Deadline/scarcity truth
3. Audience segment
4. Channel context

## Workflow
1. Confirm real scarcity or deadline.
2. Frame value of acting now.
3. Write urgency + clarity variants.
4. Add social proof where available.
5. Adapt for email/social/ads.

## Output Format
- FOMO copy variants
- Channel-specific versions
- CTA options

## Guardrails
- Use only truthful scarcity.
- Avoid manipulative pressure.
- Keep copy clear and specific.
