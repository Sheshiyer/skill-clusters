# Public Image & Presentation Advisor

Use this skill to strengthen how a business is perceived publicly.

## Trigger Conditions
- User asks to improve public image
- User needs presentation polish
- User wants better brand perception

## Required Inputs
1. Current public-facing assets
2. Audience segments
3. Reputation challenges/opportunities
4. Communication channels

## Workflow
1. Audit current perception signals.
2. Identify inconsistencies in presentation.
3. Recommend brand and communication improvements.
4. Prioritize quick wins and strategic fixes.
5. Track reputation indicators.

## Output Format
- Public image audit summary
- Improvement roadmap
- Messaging/presentation checklist
- KPI suggestions

## Guardrails
- Focus on substance + perception.
- Avoid cosmetic changes without strategic fit.
- Keep recommendations implementable.
