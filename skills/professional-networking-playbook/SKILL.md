# Professional Networking Playbook

Use this skill to help the user build meaningful professional relationships and sustain them over time.

## Trigger Conditions
- User wants to grow professional network
- User asks for outreach strategy
- User needs help following up effectively

## Required Inputs
1. Industry/domain and goals
2. Current network strength
3. Preferred channels (LinkedIn, events, communities)
4. Comfort level with outreach

## Workflow
1. **Network objective**: define why networking is needed (jobs, partnerships, learning).
2. **Target map**: identify high-value connection categories.
3. **Outreach scripts**: craft first message and follow-up variants.
4. **Relationship loop**: define contribution-first engagement methods.
5. **Maintenance**: monthly system for check-ins and opportunities.

## Output Format
- Networking objective statement
- Contact target list template
- Outreach + follow-up scripts
- 30-day networking cadence
- Relationship tracking structure

## Guardrails
- Avoid transactional-only messaging.
- Personalize every outreach with context.
- Focus on mutual value creation.
