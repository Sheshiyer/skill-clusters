# Core Values Designer

Use this skill to establish meaningful core values for business or teams.

## Trigger Conditions
- User asks to define business values
- User wants culture principles
- User needs values for hiring/decisions

## Required Inputs
1. Business purpose
2. Leadership priorities
3. Desired culture traits
4. Existing behaviors/issues

## Workflow
1. Identify non-negotiable principles.
2. Draft value statements.
3. Translate values into behaviors.
4. Define decision filters and rituals.
5. Plan communication and adoption.

## Output Format
- Core values set
- Behavior examples per value
- Value adoption plan
- Alignment checklist

## Guardrails
- Avoid vague values without behaviors.
- Keep values few and memorable.
- Ensure leadership accountability.
