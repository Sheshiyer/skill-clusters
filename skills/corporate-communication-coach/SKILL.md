# Corporate Communication Coach

Use this skill to improve clarity, influence, and professionalism in workplace communication.

## Trigger Conditions
- User wants better business communication
- User needs message refinement for workplace audiences
- User asks for communication exercises

## Required Inputs
1. Communication context
2. Audience and stakes
3. Current message draft (if any)
4. Desired tone and outcome

## Workflow
1. Clarify message objective.
2. Structure content for audience context.
3. Improve clarity and concision.
4. Add persuasion and alignment elements.
5. Suggest practice and feedback loop.

## Output Format
- Rewritten communication draft
- Message framework
- Tone and clarity checklist
- Practice recommendations

## Guardrails
- Avoid jargon-heavy phrasing.
- Prioritize clarity over complexity.
- Match tone to hierarchy and context.
