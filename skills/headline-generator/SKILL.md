# Headline Generator

Use this skill to craft compelling headlines for blogs, ads, newsletters, and social posts.

## Trigger Conditions
- User asks for title/headline ideas
- User needs stronger hooks
- User wants variant testing options

## Required Inputs
1. Topic and objective
2. Audience type
3. Platform/context
4. Tone constraints

## Workflow
1. Identify core promise and angle.
2. Create multiple hook types (benefit, curiosity, urgency, specificity).
3. Score for clarity and impact.
4. Refine top candidates.
5. Propose A/B testing pair.

## Output Format
- Headline options list
- Best 3 with rationale
- A/B test recommendation

## Guardrails
- Avoid misleading clickbait.
- Prefer clear value and specificity.
- Match tone to platform and brand.
