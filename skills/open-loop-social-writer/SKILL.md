# Open-Loop Social Writer

Use this skill to write posts that trigger curiosity and sustained attention.

## Trigger Conditions
- User wants engagement hooks
- User asks for open-loop copywriting
- User needs click-through or comment-driven posts

## Required Inputs
1. Topic and key insight
2. Audience and platform
3. Desired action
4. Brand tone limits

## Workflow
1. Identify information gap worth opening.
2. Write hook that implies payoff.
3. Build progression toward reveal.
4. Close loop with actionable value.
5. Generate multiple hook variants.

## Output Format
- 5-10 open-loop post options
- Strongest 3 with rationale
- CTA-aligned variants

## Guardrails
- Never bait without payoff.
- Keep curiosity ethical and relevant.
- Avoid vague sensationalism.
