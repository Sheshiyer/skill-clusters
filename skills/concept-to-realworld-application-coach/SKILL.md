# Concept to Real-World Application Coach

Use this skill to help users apply ideas/frameworks to practical situations.

## Trigger Conditions
- User asks how a concept applies in real life
- User wants practical examples and use cases
- User needs implementation guidance from theory

## Required Inputs
1. Concept/topic
2. User context/domain
3. Desired application area
4. Constraints

## Workflow
1. Explain concept in plain language.
2. Map concept to real scenarios.
3. Generate practical examples.
4. Build step-by-step implementation plan.
5. Add measurement and iteration guidance.

## Output Format
- Concept summary
- Real-world application examples
- Action plan
- Success indicators

## Guardrails
- Avoid abstract-only explanation.
- Match examples to user context.
- Keep steps actionable and testable.
