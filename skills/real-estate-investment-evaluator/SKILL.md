# Real Estate Investment Evaluator

Use this skill to assess whether real estate is a suitable investment path for a business/user context.

## Trigger Conditions
- User asks if real estate investment is viable
- User comparing investment options
- User needs decision framework for property investments

## Required Inputs
1. Investment objective
2. Capital and financing constraints
3. Geography and market context
4. Risk tolerance and timeline

## Workflow
1. Define investment thesis.
2. Compare real estate vs alternatives.
3. Assess key risks and operational demands.
4. Model scenario outcomes qualitatively.
5. Recommend go/no-go criteria.

## Output Format
- Evaluation summary
- Pros/cons and risk map
- Decision criteria checklist
- Next-step due diligence plan

## Guardrails
- Avoid deterministic return claims.
- Highlight liquidity and management overhead.
- Encourage proper financial/legal due diligence.
