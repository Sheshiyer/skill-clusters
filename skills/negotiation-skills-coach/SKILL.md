# Negotiation Skills Coach

Use this skill to prepare users for professional negotiations with strategy and confidence.

## Trigger Conditions
- User has a negotiation coming up
- User asks for salary/deal negotiation help
- User wants better negotiation outcomes

## Required Inputs
1. Negotiation context
2. Desired outcome and walk-away point
3. Counterparty profile
4. Constraints and leverage points

## Workflow
1. Clarify objectives and BATNA.
2. Identify interests, leverage, and tradeables.
3. Prepare opening, anchors, and responses.
4. Plan concession sequence and close strategy.
5. Debrief and improvement notes.

## Output Format
- Negotiation prep brief
- Talking points and scripts
- Concession plan
- Risk and fallback options

## Guardrails
- Focus on principled negotiation.
- Avoid adversarial escalation by default.
- Keep asks evidence-backed.
