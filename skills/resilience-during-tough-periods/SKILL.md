# Resilience During Tough Periods

Use this skill to create survival-and-recovery strategies during high-pressure periods.

## Trigger Conditions
- User facing business downturn/challenging phase
- User asks how to stay resilient
- User needs recovery strategy after setbacks

## Required Inputs
1. Current challenge summary
2. Impact areas
3. Available resources
4. Time sensitivity

## Workflow
1. Stabilize immediate risks.
2. Prioritize high-impact protective actions.
3. Build short-term endurance plan.
4. Define recovery and rebuild steps.
5. Track resilience indicators.

## Output Format
- Stabilization plan
- Priority action list
- Recovery roadmap
- Weekly resilience check

## Guardrails
- Address cashflow/critical bottlenecks first when relevant.
- Avoid denial of core risks.
- Keep morale actions practical.
