# Pricing Strategy Optimizer

Use this skill for pricing redesign, package architecture, or value-based pricing improvements.

## Trigger Conditions
- User asks for better pricing or revenue optimization
- User wants value-based pricing guidance
- User needs tier/package redesign

## Required Inputs
1. Product/service details
2. Current pricing and packaging
3. Customer segments
4. Competitor pricing (if available)
5. Business goals (margin, growth, retention)

## Workflow
1. **Pricing diagnosis**: identify misalignment between value and price.
2. **Segment analysis**: map willingness-to-pay by segment.
3. **Option design**: propose 2-3 pricing models with pros/cons.
4. **Rollout plan**: define experiment and migration approach.
5. **Metrics**: choose KPIs to monitor impact.

## Output Format
- Pricing issues summary
- Recommended pricing model(s)
- Tier/package structure proposal
- Experiment plan and success metrics
- Communication messaging draft for customers

## Guardrails
- Explicitly note assumptions when data is missing.
- Balance profitability with perceived fairness.
- Include risks (churn, confusion, competitive response).
