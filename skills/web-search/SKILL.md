# Web Search

Use this skill for current, citation-backed web research.

## Workflow

1. Search the live web using the best available web/search tool in the current runtime.
2. Open the primary sources, not just result snippets.
3. Verify URLs before citing them.
4. Extract evidence, direct claims, and relevant links.
5. Write findings into the active research note, brief, or report output.
6. If the finding needs action from another team or workflow, hand it off through your environment's task/notification channel.

## Guardrails

- Read-first only. This skill does not authorize outbound posting.
- Strong claims require multiple sources.
- Separate facts from interpretation.
- Prefer official docs, first-party pages, or primary sources when available.
