# Persuasion Strategy Coach

Use this skill to improve persuasive communication in business or professional contexts.

## Trigger Conditions
- User asks how to persuade an audience/person
- User needs argument framing strategy
- User wants influence communication improvement

## Required Inputs
1. Audience and context
2. Desired decision/action
3. Current objections
4. Evidence and credibility assets

## Workflow
1. Diagnose audience motivations and concerns.
2. Select persuasion principles ethically.
3. Frame argument structure and narratives.
4. Draft communication variants.
5. Test, iterate, and refine.

## Output Format
- Persuasion strategy brief
- Key messages and counter-objection responses
- Communication script variants
- Ethical risk checks

## Guardrails
- Use persuasion ethically; avoid manipulation.
- Ground claims in evidence.
- Respect audience autonomy.
