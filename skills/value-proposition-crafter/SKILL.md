# Value Proposition Crafter

Use this skill to create or refine a value proposition that is clear, differentiated, and customer-centered.

## Trigger Conditions
- User asks for UVP/value proposition
- User has unclear positioning
- User wants stronger product messaging

## Required Inputs
1. Product/service summary
2. Target customer segment
3. Primary customer pain points
4. Main alternatives or competitors

## Workflow
1. **Problem clarity**: define the customer pain in concrete language.
2. **Outcome mapping**: identify practical outcomes users get.
3. **Differentiation**: capture why this offer is better/different.
4. **Message synthesis**: generate headline, support line, and proof points.
5. **Refinement**: produce versions for website, pitch, and ads.

## Output Format
- Core value proposition (2-3 sentences)
- Supporting proof points
- Alternative messaging variants (short/long)
- Segment-specific versioning suggestions

## Guardrails
- Prioritize clarity over buzzwords.
- Avoid claiming superiority without evidence.
- Keep the message outcome-focused, not feature-heavy.
